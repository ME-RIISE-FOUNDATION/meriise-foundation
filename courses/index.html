MICRO-ENGINEERING CERTIFICATION PROGRAM

DEPT. OF CSE & ME-RIISE, Malnad College of Engneering,Hassan

The Micro Engineering Certification Program is intended to predominantly enhance skill set in a specific technical area and receive significant credentials. This certification program is jointly offered by Department of Computer Science and Engineering and ME-RIISE (Malnad Enclave for Research, Innovation, Incubation, Startups and Entrepreneurship) , Malnad College of Engineering, Hassan in collaboration with reputed Academic Institutes and Industries. Objectives

  • To expose learners to a field at strategic level giving requisite knowledge necessary to advance career.
  • Allow learners to gain expertise in the emerging and fastest growing areas of computer science and allied fields of engineering

The course is offered to students (engineering/diploma/any degree), faculty members, job seekers, employees seeking career advancement by Industrial experts/ Academicians from reputed universities

Every program has its own duration:
CORE JAVA: AN INDUSTRIAL APPROACH - 120 hours

Course delivery & Certification

These courses include offline lectures with hands-on, online video lectures, quizzes, programming assignments and end of course examination. Learners are required to complete all the modules specified in the respective course and achieve minimum grade. Minimum grade will be specified by the institution.

Courses Offered

CORE JAVA- AN INDUSTRIAL APPROACH

This is comprehensive certification course in Core Java that will make you grow in your software coding career. This course includes the basics of Java, Java statements, exception handling, Objects, Classes and so on. Resource Person: R.Praveen Chandra Goud, Senior Java Consultant. More details

Artificial Intelligence & Machine Learning with Hardware Implementation

This course provides a comprehensive introduction and focuses on data analytics and machine learning techniques, practical deep learning using MATLAB®. The course demonstrates the use of unsupervised learning to discover features in large data sets and supervised learning to build predictive models. Attendees will learn how to create, train, and evaluate different kinds of deep neural networks. They will also learn how deploy algorithms on hardware like GPU and FPGAs. This course seeks to fill all those gaps in knowledge that scare off beginners and simultaneously apply your knowledge of Data Science and Deep Learning to real-world business problems. More details

Practical & Real World Deep Learning

This course provides a comprehensive introduction and focuses on few concepts of machine learning techniques and practical deep learning using MATLAB® and Python. This course provides a comprehensive introduction to practical deep learning using MATLAB®. Attendees will learn how to create, train, and evaluate different kinds of deep neural networks. The training will also cover how to implement algorithms of NVIDIA GPUs to accelerate network training. More details

Fundamentals of Matlab & Simulink in Engineering Applications

This course provides a comprehensive introduction to the MATLAB® technical computing environment. No prior programming experience or knowledge of MATLAB is assumed. Themes of data analysis, visualization, modeling, and programming are explored throughout the course.
Topics include: Working with the MATLAB user interface, Entering commands and creating variables, Analyzing vectors and matrices, Visualizing vector and matrix data, Working with data files, Working with data types, Automating commands with scripts, Writing programs with branching and loops, Writing functions.
